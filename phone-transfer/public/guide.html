🚀 使用步骤

⚡ 前提条件：手机和电脑必须连接在同一个 WiFi 网络下！
1

解压文件

将 phone-transfer-mac.zip（或 phone-transfer-mac-arm.zip）解压到任意位置。

💡 如何选择版本？
• Intel 芯片 (2020年及之前的 Mac)：使用 phone-transfer-mac.zip
• Apple Silicon (M1/M2/M3/M4 芯片)：使用 phone-transfer-mac-arm.zip
点击左上角 → 关于本机 → 可以查看芯片类型
2

双击「start.command」

在解压后的文件夹中，双击「start.command」文件。

⚠️ 首次运行安全提示
如果提示"无法打开"：右键点击「start.command」→ 选择「打开」→ 在弹窗中点击「打开」
或在终端中执行：chmod +x phone-transfer start.command && ./start.command
3

浏览器自动打开本页面

程序启动后会自动打开此使用说明页面，同时终端窗口会显示局域网地址和二维码。

4

手机扫描二维码

用手机扫描上方二维码，打开上传页面，选择照片或视频即可传输到电脑。

❓ 常见问题

Q：手机扫码后无法打开页面？

A：请确认手机和电脑连接在同一个 WiFi 下。如果使用了 VPN，请先关闭。

Q：macOS 提示"无法验证开发者"？

A：右键点击「start.command」→ 选择「打开」→ 在弹窗中点击「打开」。

Q：Windows 提示"已保护你的电脑"？

A：点击「更多信息」→ 点击「仍要运行」即可。

Q：如何查看电脑的 IP 地址？

A：启动程序后终端会自动显示。也可以手动查看：

• macOS：系统设置 → Wi-Fi → 详细信息 → IP 地址
• Windows：设置 → 网络和Internet → Wi-Fi → 属性 → IPv4 地址

Q：如何停止服务？

A：关闭终端窗口即可，或按 Ctrl + C。

Q：鸿蒙手机如何扫码？

A：打开鸿蒙自带浏览器的扫码功能，或使用下拉通知栏中的"扫一扫"。
